Defined in Cadreur.js:539

Add the box in the target container.

The target container could be a child. This function is recursive.

A function could be called when the box is added. The container is passed by parameters, and the box isn't added automatically.

Parameters:

  • box HTMLElement | CadreurContainer

    The box to add

  • container CadreurContainer

    The container in which add the box

  • action Function | Null

    An optionnal function to execute when the box is added
